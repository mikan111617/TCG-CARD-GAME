# カードゲーム救済プロジェクト　

プロジェクトについて
本プロジェクトはゲーム作りを行っていて途中で断念した全ての人を代表して作成しました。 
Unityというツールを用いて作成していたゲームを放置していたのですが、一人で作るよりも、皆でアイデアを自由に好きな時に反映できたら面白いゲームができるのではないかという思いから実装しています。

開発要件
ゲームの開発にはUnityが必須です。 また、再配布禁止の著作物の配布目的ではないため、一部素材をお借りしていますが、それらを流用して別のコンテンツに利用することは禁止です。

プロジェクトルール
基本的には自由に改変および開発していただいて大丈夫です。 
以下は禁止事項です。 
・差別や個人あるいは団体にする侮蔑などの行為 
・プロジェクトを個人の所有として利用
